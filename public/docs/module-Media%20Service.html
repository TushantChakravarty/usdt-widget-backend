(static) saveMediaFile(file, user, type, save_dir) → {Promise.<Object>}

Saves a media file to the server's disk and database.
Parameters:
Name Type Description
file Object The file to be saved. {url} if type is url
user Object The user who created and is associated with the file. if the role is admin, media.user_id will be null.
type string The type of file to be saved. Can be "file" or "url". File is of type Object and URL is of type string.
save_dir string The directory to save the file to. Directory should be inside public folder. Default is "media". Example: "media", "uploads", "avatars", "documents", etc. This will also be used to generate the URL of the file. Example: /public/media/0001/abc123.jpg
